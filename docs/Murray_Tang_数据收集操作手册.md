**版本：** 2.0  
**制定日期：** 2026年3月29日

**目标：** 建立专业篮球运动员发展数据档案

## 二、各数据类别详细收集方法

### 1. 身体测量数据

#### 1.1 身高测量
**频率：** 每月1次（建议每月1日晨起）  
**工具：** 标准身高尺或墙壁标尺  
**方法：**
1. 测量时间：早晨起床后1小时内（脊柱未受压）
2. 被测者：赤脚、脱帽、穿轻薄衣物
3. 站立姿势：脚跟并拢，背靠身高尺，挺胸抬头，目视前方
4. 测量者：用直角尺轻压头顶，读取数值
5. 记录：精确到0.1cm
6. 测量3次，取平均值

**注意事项：**
- 同一测量者、同一测量工具、同一时间段
- 测量前避免剧烈运动
- 记录时注明测量日期和时间

#### 1.2 体重测量
**频率：** 每日晨起  
**工具：** 电子体重秤（精度0.01kg）  
**方法：**
1. 测量时间：晨起排便后、早餐前
2. 被测者：穿轻薄内衣或裸体
3. 秤放置：平坦坚硬地面，避免地毯
4. 测量：站稳后待数值稳定再读取
5. 记录：精确到0.01kg

**注意事项：**
- 每日同一时间测量
- 避免运动后、餐后、大量饮水后测量
- 使用同一台秤

#### 1.3 臂展测量
**频率：** 每季度1次  
**工具：** 皮尺或专用测量尺  
**方法：**
1. 被测者：靠墙站立，双臂水平侧平举
2. 双手：掌心向前，手指伸直
3. 测量：测量两中指指尖之间的距离
4. 记录：精确到0.1cm
5. 测量3次，取平均值

#### 1.4 站立摸高
**频率：** 每月1次  
**工具：** 摸高器或墙壁标尺  
**方法：**
1. 被测者：靠墙站立，双脚并拢
2. 动作：单手（优势手）最大幅度上伸
3. 测量：记录中指指尖达到的最高点
4. 记录：精确到0.1cm
5. 测量3次，取最高值

#### 1.5 手长/手宽、足长/足宽
**频率：** 每季度1次  
**工具：** 直尺或游标卡尺  
**方法：**
- 手长：手腕横纹至中指指尖
- 手宽：拇指外展时，拇指尖至小指根
- 足长：脚跟至最长脚趾尖
- 足宽：足掌最宽处

### 2. 身体成分测量

#### 2.1 体脂率测量
**频率：** 每月1次  
**工具：** 生物电阻抗体脂秤或皮脂钳  
**方法（生物电阻抗）：**
1. 测量时间：晨起空腹、排尿后
2. 被测者：赤脚站立，双脚接触电极
3. 输入：年龄、性别、身高
4. 测量：等待仪器读数稳定
5. 记录：体脂率%、肌肉量kg、BMI

**注意事项：**
- 避免运动后、餐后、大量饮水后测量
- 女性注意生理周期影响
- 使用同一台设备

#### 2.2 BMI计算
**公式：** BMI = 体重(kg) / 身高²(m)  
**自动计算：** 系统根据身高体重自动计算  
**参考标准（10岁男童）：**
- 偏瘦：<14.2
- 正常：14.2-19.4
- 超重：19.4-21.4
- 肥胖：>21.4

### 3. 爆发力测试

#### 3.1 原地纵跳 (Squat Jump, SJ)
**频率：** 每季度1次  
**工具：** 纵跳测试垫或Vertec摸高器  
**方法：**
1. 准备：被测者站立，双脚与肩同宽
2. 起始姿势：双手叉腰，屈膝90°（大腿与地面平行）
3. 动作：听到指令后立即向上跳起，不得预摆
4. 落地：双脚同时落地，保持平衡
5. 测量：记录离地高度（cm）
6. 测试3次，取最高值

**注意事项：**
- 起跳前不得有预摆动作
- 双手始终叉腰，不得摆动
- 起跳角度垂直向上

#### 3.2 反向跳 (Countermovement Jump, CMJ)
**频率：** 每季度1次  
**工具：** 纵跳测试垫或Vertec摸高器  
**方法：**
1. 准备：被测者自然站立
2. 动作：快速下蹲（约90°）后立即向上跳起
3. 摆臂：可自然摆臂助力
4. 测量：记录离地高度（cm）
5. 测试3次，取最高值

**与SJ的区别：** CMJ允许预摆和摆臂，更接近实战

#### 3.3 助跑纵跳 (Max Vertical Jump)
**频率：** 每季度1次  
**工具：** Vertec摸高器或纵跳测试垫  
**方法（NBA标准）：**
1. 准备：被测者从助跑线后开始
2. 助跑：自由助跑（通常2-4步）
3. 起跳：单脚或双脚起跳
4. 摸高：单手触碰最高点
5. 测量：记录最高触碰点（cm）
6. 计算：纵跳高度 = 助跑摸高 - 站立摸高
7. 测试3-5次，取最高值

**注意事项：**
- 助跑距离不限，以舒适为准
- 可以单脚或双脚起跳
- 充分热身后再测试

#### 3.4 立定跳远
**频率：** 每季度1次  
**工具：** 跳远垫或沙坑+卷尺  
**方法：**
1. 起跳线：双脚不得越线
2. 动作：原地摆臂起跳，向前跳出
3. 落地：双脚同时落地，保持平衡
4. 测量：从起跳线至最近落地点（脚跟）
5. 记录：精确到1cm
6. 测试3次，取最高值

### 4. 速度测试

#### 4.1 20米冲刺
**频率：** 每季度1次  
**工具：** 电子计时门或秒表  
**方法：**
1. 起点：站立式起跑，前脚距起跑线约30cm
2. 计时：前脚通过起跑线开始计时
3. 终点：身体躯干通过终点线停止计时
4. 测试：至少2次，取最快成绩
5. 休息：每次测试间休息3-5分钟

**注意事项：**
- 充分热身，特别是下肢
- 起跑反应要快但不要抢跑
- 全速冲刺通过终点

#### 4.2 3/4场地冲刺 (NBA标准)
**频率：** 每季度1次  
**工具：** 电子计时门  
**方法：**
1. 距离：约22.86米（75英尺）
2. 起点：站立式起跑
3. 终点：通过终点线
4. 记录：精确到0.01秒

#### 4.3 10米加速测试
**频率：** 每半年1次  
**方法：**
1. 测量前10米分段计时
2. 评估加速能力
3. 可与20米冲刺结合测试

### 5. 敏捷性测试

#### 5.1 T型跑 (T-Test)
**频率：** 每季度1次  
**工具：** 秒表、4个标志桶  
**场地设置：**
```
    A (10米)
    |
B --+-- C (5米)
    |
    D (起点)
```
- A点：D点正前方10米
- B、C点：A点左右各5米

**方法：**
1. 起点：D点站立式起跑
2. 路线：D→A→B→C→A→D
3. 要求：必须触碰每个点
4. 计时：从起跑至返回D点
5. 测试2-3次，取最快成绩

#### 5.2 5-10-5折返跑 (Pro Agility Test)
**频率：** 每季度1次  
**工具：** 秒表、3个标志桶  
**场地设置：**
- 3个标志桶排成一线，间距5码(4.57米)
- 中间桶为起点

**方法：**
1. 起点：中间桶，面向侧面
2. 路线：向左/右跑5码→折返跑10码→折返跑5码回起点
3. 计时：从启动至通过终点
4. 左右各测1次，取平均或最快

#### 5.3 车道敏捷测试 (Lane Agility - NBA标准)
**频率：** 每半年1次  
**工具：** 秒表、标志桶  
**场地设置：**
- 长方形区域，尺寸根据年龄调整
- 标准NBA：宽4.57米×长5.79米

**方法：**
1. 按指定路线跑动（滑步、后退跑、冲刺组合）
2. 计时完成全程
3. 测试2次，取最快

### 6. 耐力测试

#### 6.1 Yo-Yo间歇恢复测试
**频率：** 每季度1次  
**工具：** 录音/APP、标志桶、平坦场地(20米)  
**方法：**
1. 设置：两条线相距20米
2. 节奏：按录音提示往返跑
3. 间歇：每次往返后有10秒休息
4. 速度：速度逐级增加
5. 结束：两次未能按时到达终点
6. 记录：完成的级别和总距离

**参考标准（10岁）：**
- 优秀：级别15以上
- 良好：级别12-14
- 一般：级别10-11
- 需提高：级别10以下

#### 6.2 折返跑 (Shuttle Run)
**频率：** 每月1次  
**方法：**
1. 设置：两条线相距20米
2. 时间：设定时间（如1分钟）
3. 计数：记录完成的往返次数
4. 或：设定次数，记录完成时间

#### 6.3 心率恢复测试
**频率：** 每月1次  
**工具：** 心率监测器  
**方法：**
1. 运动后：达到最大心率（或170bpm以上）
2. 停止运动，立即坐下
3. 测量：1分钟后心率
4. 计算：心率下降值 = 运动后心率 - 1分钟后心率
5. 评估：下降>30bpm为良好

### 7. 力量测试

#### 7.1 卧推测试 (NBA标准：84kg/185lbs)
**频率：** 每年1次（15岁后开始）  
**工具：** 杠铃、卧推架  
**方法：**
1. 重量：84kg（青少年可用适当减重）
2. 动作：标准卧推动作
3. 记录：最大重复次数（不借力）
4. 安全：必须有保护员

**青少年替代方案：**
- 俯卧撑最大次数测试
- 或自身体重卧推

#### 7.2 深蹲1RM测试
**频率：** 每半年1次（需专业指导）  
**方法：**
1. 充分热身
2. 递增重量，每次增加5-10kg
3. 每组1-3次
4. 记录能完成1次的最大重量
5. 必须有专业保护和指导

**青少年替代方案：**
- 深蹲最大次数（自重或轻重量）
- 单腿深蹲测试

#### 7.3 握力测试
**频率：** 每季度1次  
**工具：** 握力计  
**方法：**
1. 左右手分别测试
2. 站立，手臂自然下垂
3. 全力握紧握力计
4. 每只手测试3次，取最高值
5. 记录左右手数据

#### 7.4 俯卧撑测试
**频率：** 每月1次  
**方法：**
1. 标准俯卧撑姿势
2. 1分钟内最大次数
3. 胸部触地，手臂完全伸直
4. 保持身体直线

#### 7.5 平板支撑
**频率：** 每月1次  
**方法：**
1. 前臂支撑姿势
2. 身体成一条直线
3. 记录能保持的时间
4. 臀部不得上翘或下沉

### 8. FMS功能性动作筛查

#### 8.1 测试频率
- **常规：** 每月1次
- **伤后恢复：** 每周1次（康复期）
- **赛季前：** 必须测试

#### 8.2 测试环境
- 平坦、安静的场地
- 充足的空间
- 测试垫（可选）
- FMS测试工具包（可选）

#### 8.3 7项基础测试评分标准

**评分标准：**
- **3分：** 动作完美，无代偿
- **2分：** 能完成动作，但有代偿
- **1分：** 无法完成动作
- **0分：** 测试过程中出现疼痛

**测试项目：**

1. **深蹲 (Deep Squat)**
   - 站立，双脚与肩同宽，脚尖向前
   - 双臂上举，与地面垂直
   - 深蹲至大腿低于水平面
   - 保持脚跟不离地，膝盖不外翻

2. **跨栏步 (Hurdle Step)**
   - 站立，双脚并拢
   - 跨过一个高度至胫骨粗隆的栏架
   - 保持平衡，无身体晃动
   - 左右各测，取低分

3. **直线弓步 (In-Line Lunge)**
   - 前后脚成一直线，间距为小腿长度
   - 后膝轻触地面
   - 保持躯干直立
   - 左右各测，取低分

4. **肩部灵活性 (Shoulder Mobility)**
   - 一手从肩上方向下伸，一手从背后向上伸
   - 测量两手间距
   - 左右各测，取低分

5. **主动直腿上抬 (ASLR)**
   - 仰卧，一腿伸直上抬
   - 另一腿保持贴地
   - 测量抬起角度或垂直距离
   - 左右各测，取低分

6. **躯干稳定俯卧撑 (TSPU)**
   - 俯卧，双手置于肩部下方
   - 俯卧撑姿势，保持身体直线
   - 根据能力选择起始高度

7. **旋转稳定性 (Rotary Stability)**
   - 四点支撑姿势
   - 对侧手脚同时伸展
   - 保持躯干稳定
   - 左右各测，取低分

#### 8.4 疼痛排查测试
- 肩部清理测试
- 脊柱伸展清理
- 脊柱屈曲清理

**如出现疼痛，该测试项得0分，并建议就医检查**

#### 8.5 结果解读
- **21分：** 完美
- **17-20分：** 良好，低风险
- **14-16分：** 一般，中等风险
- **<14分：** 高风险，建议干预

### 9. 投篮测试

#### 9.1 罚球测试
**频率：** 每周1次  
**方法：**
1. 10次罚球
2. 标准罚球姿势
3. 记录命中次数
4. 计算命中率

#### 9.2 定点投篮测试
**频率：** 每月1次  
**测试点：**
- 右翼中距离
- 弧顶中距离
- 左翼中距离
- 右翼三分
- 弧顶三分
- 左翼三分
- 右底角三分
- 左底角三分

**方法：**
1. 每个点位10次出手
2. 自投自抢或有人传球
3. 记录命中次数
4. 计算各区域命中率

#### 9.3 投篮热图制作
**频率：** 每季度1次  
**方法：**
1. 在球场图上标注8个区域
2. 每个区域测试10-20次
3. 记录命中次数
4. 用颜色标注命中率
   - 绿色：>60%
   - 黄色：40-60%
   - 红色：<40%

### 10. 运球测试

#### 10.1 直线运球速度
**频率：** 每月1次  
**方法：**
1. 全场(28米)直线运球
2. 分别测试左右手
3. 计时从起点到终点
4. 测试3次，取最快

#### 10.2 变向运球
**频率：** 每月1次  
**方法：**
1. 设置5-6个标志桶，间距2-3米
2. 绕桩变向运球通过
3. 计时完成全程
4. 测试3次，取最快

#### 10.3 高级运球成功率
**频率：** 每季度1次  
**方法：**
1. 背后运球：连续10次，记录成功次数
2. 胯下运球：连续10次，记录成功次数
3. 转身运球：连续10次，记录成功次数
4. 计算成功率

## 三、数据收集规范与注意事项

### 3.1 数据质量原则

**准确性：**
- 使用校准的测量工具
- 严格按照标准方法操作
- 多次测量取平均值

**一致性：**
- 同一测量者
- 同一测量工具
- 同一时间段
- 同一环境条件

**完整性：**
- 按时收集所有规定数据
- 不遗漏必填项
- 异常数据注明原因

### 3.2 测试前准备

**被测者准备：**
- 充分热身（10-15分钟）
- 避免测试前24小时内大强度训练
- 正常饮食和睡眠
- 穿着合适运动装备

**环境准备：**
- 适宜的温度（15-25°C）
- 良好的照明
- 安全的场地
- 必要的测试设备

### 3.3 数据记录规范

**记录要求：**
- 立即记录，避免遗忘
- 字迹清晰（手写时）
- 使用标准单位
- 异常数据标注原因

**数据备份：**
- 每日同步到系统
- 定期导出备份
- 保留原始记录

### 3.4 安全注意事项

**测试安全：**
- 充分热身
- 循序渐进
- 有保护员（力量测试）
- 不适立即停止

**伤病预防：**
- FMS<14分时降低训练强度
- 疲劳时避免大强度测试
- 伤后恢复测试需医生许可

### 3.5 特殊情况处理

**数据缺失：**
- 注明缺失原因
- 尽量补测
- 分析时说明

**异常数据：**
- 检查是否为测量错误
- 确认后保留并标注
- 分析时考虑

**伤病期间：**
- 暂停体能测试
- 可进行恢复监测
- 伤愈后重新建立基线
